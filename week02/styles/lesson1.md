## Media Query Changes Summary (week02)

### styles/media-query.css
- Base styles set without media queries.
- Body uses Roboto Google Font.
- Header, main, and footer:
  - Max width 640px, centered with margin.
  - Faint blue border, padding, blueish background.
- Main is a single-column CSS grid with 1rem gap.
- Section elements have padding and a lighter background.
- Descriptive comments added for each property.

### styles/media-query-large.css
- Media query for viewport width 500px or greater:
  - Header background changes to black, text to white.
  - Main displays two equal columns (grid-template-columns: 1fr 1fr).
- Comments included for clarity.
