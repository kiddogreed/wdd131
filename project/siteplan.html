Site Name

John D's Web Gatherer

"John D's Web Gatherer" is a personal, hands-on learning resource built entirely with HTML, CSS, and JavaScript — no frameworks, no TypeScript, no Python — demonstrating mastery of core front-end fundamentals. The name reflects the author's identity (John D.) and the site's mission: teaching how to gather data from the web. Future versions may expand with Python or frameworks, but this release is intentionally pure HTML/CSS/JS.

Optional domain: jd-webgatherer.dev

Site Purpose

John D's Web Gatherer is a beginner-friendly educational website focused on static web scraping. The site explains what static web scraping is, how it works, and the tools commonly used — such as Python's requests and BeautifulSoup libraries. It provides simple, annotated code examples, best practices for organizing scraped data (CSV, JSON), and a clear overview of ethical and legal considerations so newcomers can scrape responsibly.

Scenarios

  • Scenario 1: "I've heard about web scraping but don't know where to start. What tools do I need and how do I write my first scraper?"
  • Scenario 2: "Is web scraping legal? What ethical rules should I follow before scraping someone's website?"
  • Scenario 3: "I scraped some data but now it's one big mess. How do I clean and save it in an organized way?"
